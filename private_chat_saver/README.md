# Telegram Private Chat Saver

Личный локальный архиватор переписки Telegram. Подключается к **вашему**
аккаунту через Telethon (MTProto) после вашей авторизации — это **не**
BotFather-бот. В реальном времени сохраняет:

- новые сообщения из личных чатов (входящие и исходящие),
- фото, видео, кружки, голосовые, документы → в `media/`,
- правки (оригинальный текст остаётся),
- **удаления**: если сейвер уже видел сообщение, при удалении оно
  помечается 🗑, но остаётся в локальной базе.

**Приватность:** всё хранится только на вашем компьютере — SQLite
(`private_saver.db`) и папка `media/`. Программа ничего никуда не отправляет
(единственное соединение — официальный MTProto-сервер Telegram, без него
получать сообщения невозможно).

## Установка

```bash
cd private_chat_saver
python3 -m venv venv
venv/bin/pip install -r requirements.txt
cp .env.example .env
```

Получите API-ключи (это ключи вашего аккаунта, не бот-токен):

1. Откройте https://my.telegram.org → войдите под своим номером.
2. «API development tools» → создайте приложение.
3. Впишите `API_ID` и `API_HASH` в `.env`.

## Запуск

```bash
venv/bin/python saver.py
```

При **первом запуске** Telethon спросит номер телефона и код подтверждения
из Telegram (и пароль 2FA, если включён). Сессия сохранится локально в
`private_saver.session` — дальше вход автоматический.

Оставьте процесс работать в фоне — он архивирует всё новое в личных чатах.

## Команды в Telegram

Пишите команды **самому себе** в чат «Saved Messages» (Избранное) —
сейвер ответит там же:

| Команда | Что делает |
|---|---|
| `/recent [N]` | последние N сохранённых сообщений (по умолчанию 10) |
| `/deleted [N]` | удалённые сообщения |
| `/search текст` | поиск по тексту, имени отправителя и названию чата |
| `/files [N]` | последние сохранённые файлы и медиа |
| `/stats` | статистика базы |
| `/help` | справка |

## Просмотр из терминала

Работает без подключения к Telegram (читает локальную базу):

```bash
venv/bin/python viewer.py recent 20
venv/bin/python viewer.py deleted
venv/bin/python viewer.py search "договор"
venv/bin/python viewer.py chats
venv/bin/python viewer.py chat "Иван"
venv/bin/python viewer.py files
venv/bin/python viewer.py stats
```

## Настройки (.env)

| Переменная | По умолчанию | Описание |
|---|---|---|
| `API_ID` / `API_HASH` | — | ключи с my.telegram.org (обязательно) |
| `SESSION_NAME` | `private_saver` | имя файла сессии |
| `MEDIA_DIR` | `media` | папка для медиа |
| `MAX_FILE_MB` | `50` | файлы больше — не скачиваются (запись о сообщении остаётся) |
| `SAVE_OUTGOING` | `true` | сохранять ли собственные исходящие |

## Структура данных

- `private_saver.db` — SQLite: текст, отправитель, чат, время, правки,
  пометки удаления.
- `media/<chat_id>/<msg_id>_имяфайла` — фото, видео, голосовые, документы.

## Ограничения

- Сохраняются только сообщения, пришедшие **пока сейвер запущен**;
  удаление ловится, только если сообщение уже было сохранено.
- Группы и каналы не сохраняются — только личные чаты (включая ботов).
- Секретные чаты (E2E) недоступны через API.

## Важно

- `*.session`, `private_saver.db`, `media/` и `.env` содержат личные данные —
  они уже исключены в `.gitignore`, не передавайте их никому.
- Инструмент предназначен только для архива вашей собственной переписки
  на вашем устройстве.
